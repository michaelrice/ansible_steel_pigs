# ansible_steel_pigs

## About

anisble_steel_pigs is an Ansible role that you can use to install [steel_pigs](https://github.com/virtdevninja/steel_pigs).

## Supported Platforms

ansible_steel_pigs has been tested on the following platforms:

* Ubuntu (14.04)
* Fedora (22)

## Example

```
---
- hosts: steel_beards
  roles:
    - ansible_steel_pigs
```
